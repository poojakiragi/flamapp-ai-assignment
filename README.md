---

## Quick push commands (once you run the script locally)

1. Create a new **empty** GitHub repository named `flamapp-ai-assignment` on your GitHub account.
2. From the generated folder run:
```bash
git remote add origin https://github.com/poojakiragi/flamapp-ai-assignment.git
git branch -M main
git push -u origin main
```

Make sure you run the script in Git Bash (Windows) or a Unix-compatible shell.
