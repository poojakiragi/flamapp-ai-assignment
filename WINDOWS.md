# Running on Windows (Git Bash)

1. Install Git for Windows: https://git-scm.com/download/win
2. Open **Git Bash** (not PowerShell) and navigate to the extracted folder.
3. Run:
   chmod +x create_repo.sh
   ./create_repo.sh
4. Then push using the commands in the script.

If you prefer PowerShell, you can run the same commands in WSL or adapt the script to PowerShell.
